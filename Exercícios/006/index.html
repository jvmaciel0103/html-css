Entendendo o posicionamento dos títulos

Introdução

Por quê isso é importante?

Para deixar seu site esteticamente organizado

Para ser atrativo e compreensivo em tópicos

Ordenamento dos títulos

Os títulos são definidos por <hx>

Útilizando assim para formar tópicos de ordem crescente

x reprenta os números acompanhados da tag <h>

Ordem de tamanho

Ápos os títulos selecionados, para escrever uma informação em forma de texto, pode se utilizar a tag <p>assim como foi feito no mesmo.
